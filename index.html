About the Project

Rapid urbanization in Indonesian cities continues to present major spatial planning challenges, including unregulated land use, scattered development, and limited oversight. Government efforts to monitor urban growth and ensure compliance with zoning plans like RTRW (Rencana Tata Ruang Wilayah / Spatial Plan) are often hindered by manual processes that are time-consuming, costly, and outdated.

This project leverages ArcGIS Pro’s Deep Learning tools to extract building footprints from aerial imagery automatically. The resulting data provides a scalable, up-to-date baseline that supports urban growth monitoring and feeds into spatial conformity evaluations, such as KKPR (Kesesuaian Kegiatan Pemanfaatan Ruang / Spatial Suitability Permit).

Once extracted, the building data is published to ArcGIS Online for collaborative exploration and decision-making. Through this GeoAI-powered workflow, the project demonstrates how automation and smart mapping technologies can enhance policy enforcement, optimize spatial analysis, and empower governments with faster, data-driven urban governance.

Building Extraction Workflow

An end-to-end geospatial workflow powered by ArcGIS — from aerial imagery acquisition to automated building detection, online publishing, and spatial conformity assessment aligned with RTRW (Spatial Plan) and KKPR (Spatial Suitability Permit) requirements.

 Data Acquisition

High-resolution aerial imagery of Depok is collected as the primary geospatial input. This imagery enables the identification of physical features such as building density.

 Deep Learning

ArcGIS Pro's Detect Objects Using Deep Learning tool is used to extract building footprints automatically, reducing manual mapping and improving processing speed and accuracy.

 Data Publishing

Extracted buildings are published through ArcGIS Online as an interactive web map, supporting open access, collaboration, and decision-making for spatial planning authorities.

 Spatial Analysis

Overlays with RTRW (Spatial Plan) boundaries allow assessment of conformity with zoning functions and inform KKPR (Spatial Suitability Permit) evaluations for urban development.

Use Case: Urban Growth & Zoning Assesment

Building footprint data extracted using ArcGIS Pro’s Deep Learning capabilities can be cross-referenced with zoning layers from the RTRW (Rencana Tata Ruang Wilayah / Spatial Plan) to evaluate compliance for KKPR (Kesesuaian Kegiatan Pemanfaatan Ruang / Spatial Suitability Permit).

Note: The overlaid building footprints have been simplified due to file size constraints. Zoning data visualized from GISTARU online viewer by ATR/BPN.

KKPR is a regulatory mechanism in Indonesia that ensures land development activities align with zoning regulations established in the RTRW or RDTR (Detailed Spatial Plan). Authorities assess whether buildings are located in suitable zones and comply with allowable land use functions.

In this use case, building footprints extracted automatically using ArcGIS Pro’s Detect Objects Using Deep Learning tool are spatially overlaid with zoning boundaries from Depok City's official spatial plan. Most extracted buildings are situated in the Kawasan Perdagangan dan Jasa (Commercial and Services Zone), and visual interpretation confirms that many of them appear to be shopping malls and retail facilities — matching the intended function.

However, a number of residential-looking structures also exist in this zone. While mixed-use zoning may allow limited residential activity, it becomes an issue when housing dominates a zone meant for commerce. Conversely, commercial buildings inside residential zones may also violate zoning policy.

With deep learning-driven building extraction, urban planners and land officials can:

  • Identify building locations relative to zoning designations.
  • Measure building area and density to assess development intensity.
  • Analyze spatial patterns to flag non-conforming land use.

This geospatial approach streamlines the KKPR process by transforming manual checks into efficient, data-driven assessments — ultimately supporting better compliance monitoring and informed urban decision-making.

Deep Learning Extraction Workflow

Below is the step-by-step configuration used to extract building footprints using ArcGIS Pro’s Deep Learning tools.

Training Samples

Digitized building footprints with a Class value of 1 are used as ground truth labels for training data preparation.

Export Training Data

Exported using RCNN Masks format via the Export Training Data tool. Tiled datasets are generated from imagery and labels.

Model Architecture

Mask R-CNN with ResNet-50 is used as the backbone model, allowing instance-based segmentation of building features.

Epoch & Processing

Training uses GPU or CPU with customizable Max Epochs to improve model accuracy and generalization.

Detection Output

Final detection is executed via the Detect Objects Using Deep Learning tool to produce vector building polygons.

Project Conclusion

This project combines geospatial AI, zoning regulations, and smart planning to propose an efficient method of monitoring building development across urban areas.

Title: Smart Urban Mapping: Deep Learning-Driven Building Extraction for Future Growth Monitoring

Background: Rapid urbanization presents challenges for governments to ensure development complies with long-term spatial plans. Traditional methods are often time-consuming, labor-intensive, and lack scalability.

Objective: To automate the detection of urban structures using Deep Learning in ArcGIS Pro and assess their conformity with RTRW (Spatial Plans) in support of KKPR (Spatial Suitability Permits).

Workflow: Starting from aerial imagery, this project uses the Detect Objects Using Deep Learning tool to generate vectorized building footprints. These features are analyzed and overlaid with RTRW zoning data, identifying areas where buildings conform—or deviate—from intended land use.

Use Case Highlight: In Depok City, buildings located in the Commercial and Services Zone were largely compliant, but some residential clusters raised zoning suitability concerns. Such insights are crucial for KKPR evaluation, which relies on spatial data like building location, size, and usage intensity.

Key Technologies: ArcGIS Pro, ArcGIS Online, ArcGIS Instant Apps. The project integrates GeoAI with spatial planning to support real-world regulatory workflows.

Impact: This approach offers a scalable, data-driven method for monitoring urban growth and assessing spatial suitability. It enables faster KKPR assessments, enhances spatial transparency, and supports smarter urban governance with Esri’s geospatial ecosystem.
